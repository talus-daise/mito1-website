## 概要

掲示板や質問箱、時間割表示など、学校生活を補助する各種機能を提供します。

公開URL  
https://talus-daise.github.io/mito1-website/

---

## 主な機能

- 時間割表示・過去一覧表示
- 名簿
- 課題
- 掲示板
- 質問箱
- リクエスト曲
- マイページ
- トラブルシューティング
